Política de Privacidade

Última atualização: [Data]

1. Introdução

Bem-vindo ao aplicativo Minhas Contas, desenvolvido por Crânio da Tecnologia. Esta política de privacidade foi criada para esclarecer aos usuários como lidamos com as informações coletadas e armazenadas pelo aplicativo. A sua privacidade é importante para nós, e comprometemo-nos a proteger qualquer informação que nos forneça.

2. Coleta e Uso de Dados

O aplicativo Minhas Contas não coleta ou armazena dados pessoais como nome, endereço, e-mail ou qualquer outra informação que possa identificar o usuário individualmente.

3. Permissões

O aplicativo solicita apenas a permissão de armazenamento para realizar o backup da base de dados no dispositivo. Essa permissão é necessária para garantir que seus dados financeiros, como registros de despesas e receitas, estejam seguros e disponíveis em caso de necessidade de restauração.

4. Armazenamento de Dados

Todos os dados inseridos no aplicativo são armazenados localmente no dispositivo do usuário. O backup de dados também é salvo apenas no armazenamento do dispositivo e não é compartilhado com terceiros ou servidores externos.

5. Segurança

Adotamos medidas de segurança para proteger as informações armazenadas no dispositivo contra acesso não autorizado, alteração, divulgação ou destruição. Contudo, é importante que o usuário também tome precauções, como evitar o compartilhamento do dispositivo com terceiros e manter o sistema operacional atualizado.

6. Alterações nesta Política de Privacidade

Esta política de privacidade pode ser atualizada periodicamente para refletir mudanças no aplicativo ou em nossas práticas de privacidade. Recomendamos que o usuário revise esta política regularmente para se manter informado sobre como protegemos suas informações.

7. Contato
